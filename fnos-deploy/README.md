# 小财迷 · 飞牛 fnOS 一键部署包

> **个人财务管理系统** — 记账 / 资产追踪 / 预算管理 / 数据分析

## 🚀 部署方式（二选一）

### 方式一：Docker Compose（推荐）

1. **上传**：将本文件夹上传到飞牛 NAS（如 `/vol1/1000/docker/xiaocaimi/`）
2. **配置**（可选）：编辑 `docker-compose.yml`，修改端口映射等
3. **部署**：
   - 打开飞牛桌面 → 「Docker」→「Compose」→「新增项目」
   - 项目名称：`xiaocaimi`
   - 存放路径：选择上传的文件夹
   - 自动识别 `docker-compose.yml` → 点击「部署」

### 方式二：命令行

```bash
# SSH 到飞牛 NAS
cd /vol1/1000/docker/xiaocaimi
docker compose up -d
```

## 📖 使用

部署成功后访问：**http://你的NASIP:8080**

| 功能 | 说明 |
|------|------|
| 记账 | 收入 / 支出 / 转账，多账户 |
| 资产 | 现金 / 银行卡 / 基金 / 负债 |
| 分析 | 月度趋势、分类占比、自动洞察 |
| 预算 | 月度总预算 + 分类预警 |
| 日历 | 事件 + 热力图 |
| 任务/提醒 | 轻量项目管理 |
| 便签/联系人/文档 | 生活资料管理 |
| 餐食计划 | 菜谱 + 周计划 + 购物清单 |

## 🔄 更新

```bash
docker compose pull
docker compose up -d
```

浏览器打开页面会显示 OTA 更新横幅，点击「立即更新」即可。

## ⚙️ 可选配置

编辑 `docker-compose.yml` 中的环境变量：

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `SYNC_TOKEN` | 多端同步密码 | 留空（仅内网） |
| `TZ` | 时区 | Asia/Shanghai |

## 📂 数据

所有数据持久化在 `./data/` 目录：
- `data.json` — 主数据
- `backups/` — 自动备份（保留最近 30 份）

导出备份：设置页 → 导出数据
